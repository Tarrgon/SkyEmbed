# SkyEmbed - Another Discord Embed Service

### Supported sites
| Site  | URL  |
|--|--|
| Bluesky | https://bskyi.app |

### Usage
Change the base URL from the supported site into the URL on the right. 
For example `https://bsky.app/profile/sfw.tarrgon.gay/post/3lntq3esbfk2p` becomes `https://bskyi.app/profile/sfw.tarrgon.gay/post/3lntq3esbfk2p` (notice the additional `i`)
